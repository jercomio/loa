## Lunar Orbiter Algorithms (Loa)

Lunar Orbiter Algorithms est une petite librairie TypeScript/JavaScript qui regroupe plusieurs fonctions algorithmiques (traitement de chaînes, permutations, horodatage, nombres d’or, rectangles de Fibonacci, helpers HTML/Markdown, etc.) utilisables côté Node.js et navigateur.

Pour la **documentation complète**, les exemples de code et les détails mathématiques, consultez :

- `README_EN.md` – **Complete documentation (English version)**
- `README_FR.md` – **Documentation complète (version française)**

Une interface web de type **Playground** est également fournie dans le dossier `Loa-Interface` pour tester visuellement la plupart des fonctions de la librairie.
